An interactive C++ desktop application built with the Qt Framework for creating and managing graphs.

Key Features:

Interactive Drawing: Add nodes with a right-click and connect them with a left-click.

Manipulation: Move nodes via drag-and-drop with collision detection.

Graph Types: Supports both directed and undirected graphs.

Data Export: Automatically generates and saves the adjacency matrix to Matrix.txt.

Tech Stack: C++17, Qt (Core, GUI, Widgets), QPainter.
